About project

This tool was created to test my skills out. But the main purpose was, obviously, the intention to help students with performing really complex operations. Converting, changing, solving - all that stuff.

Before creation

Before I started creating this thing, I had some HTML, JS and CSS coding skills. And Python, of course. I haven't thought about difficulty of creating project like this, so i've encountered some unexpected moments. So, I started to get new coding experience, learn, read and, obviously, code.

After creation

After all encounters and other challenging moments, i've got much fun, skills and experience. I learned how to use Django and Flask, advanced animations (JQuery + Skel), advanced styling (HTML5 + CSS3), request processing (AJAX) and web-administration (a bit).

Flask

The back-end of this project. First time I used Django, but Flask was more suitable here. It's easy to learn, it's incredibly powerful and encourages rapid web-development (really). Python and web-development meets there. All info is located here.

AJAX

BioTools is based on interaction of Flask and AJAX requests to make service as simple and user-friendly as possible. Documentation is located here.

Web design

Great User Interface is the main part of any project. To be understood you should speak on the language that is understood by everyone, you know. Design - is a language, that is intuitive and really understandable. By the way, this application satisfies PWA standarts.
